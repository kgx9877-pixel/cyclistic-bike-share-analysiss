## Dataset
The dataset contains bike trip data including ride duration, user type, and timestamps.

## Business Problem
How can Cyclistic convert casual riders into annual members?

## Analysis Steps
- Data Cleaning
- Data Processing
- Exploratory Data Analysis
- Visualization

## Tools & Technologies
- Python (Pandas, Matplotlib)
- Google Colab
- GitHub

## Key Insights
- Casual riders prefer weekends
- Members ride more frequently
- Ride duration is higher for casual users

## Recommendations
- Offer weekend discounts
- Promote membership plans
- Target casual riders with ads
